Masalah

Claude Code tidak jalan
di Android.

Tiga alasan kenapa install normal selalu gagal — dan kenapa workaround lama tidak cukup.

01 / 03
⚙️
Binary glibc vs Bionic

Binary resmi Anthropic dikompilasi untuk glibc Linux. Android menggunakan Bionic libc dengan ABI berbeda — binary tidak bisa jalan langsung.

02 / 03
📁
/tmp tidak tersedia

Claude Code hardcode path /tmp untuk IPC socket dan temporary files. Path ini tidak ada di Android, menyebabkan crash saat startup.

03 / 03
⏳
Versi lama = fitur terbatas

Workaround selama ini: pin ke v2.1.112 (JS only, via node cli.js). Latest version dari Anthropic sama sekali tidak bisa dijalankan.

Solusi

Native binary +
glibc-runner.

Tiga layer yang bekerja bersama untuk menjalankan binary resmi Anthropic secara native di Termux.

⚡
glibc-runner

Jalankan binary glibc resmi langsung di Android Bionic via grun — tanpa full Linux environment, tanpa proot-distro, tanpa Ubuntu.

grun · Termux pkg
🔧
Platform Patch

Auto-patch cli-wrapper.cjs & install.cjs: deteksi android → redirect ke linux. Official ARM64 binary langsung terdeteksi dengan benar.

awk · in-place patch
🚀
Latest Version

Bukan versi lama frozen. Binary ARM64 terbaru langsung dari npm @anthropic-ai/claude-code-linux-arm64, performa native penuh.

npm latest · ARM64

Install

Pilih cara install.

Tiga opsi tersedia. Shell script direkomendasikan — handle semua setup otomatis.

bash · termux
$ curl -fsSL https://raw.githubusercontent.com/DamnSit/claude-code-termux/main/install.sh | bash
ℹ Installer akan handle: pkg update, deps (nodejs-lts, glibc-runner), npm install, platform patch, API key input, alias setup. Setelah selesai: source ~/.bashrc && claude
npm · termux
$ pkg update && pkg install nodejs-lts
$ npm install -g @xurxuo/claude-code-termux
ℹ Install via NPM registry. Package wrapper yang handle setup otomatis. Sama dengan shell method tapi via npm.
bash · verified · termux
